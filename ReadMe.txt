Breed Identification System with Face AuthenticationThis project is a Tkinter-based desktop application that first performs face authentication using embeddings and then allows the user to classify images of dog and cat breeds. It integrates TensorFlow Lite models for lightweight inference and a custom face embedding pipeline for secure access.RequirementsPython Version 3.9+ (recommended)Python Libraries Used (Recommend activate virtual environment)• os (standard library)• tkinter (standard library, GUI toolkit)• Pillow (PIL) → for image handling• opencv-python (cv2) → for camera access and frame processing• random (standard library)• psutil → for system/battery information• time (standard library)• numpy → for array manipulation• tensorflow → for TensorFlow Lite inference• pickle → for loading class index mappings• dlib → for face detection• imgbeddings → for generating face embeddingsInstall required dependencies by using command below```bashpip install pillow opencv-python psutil numpy tensorflow dlib imgbeddings```Files Included• test.py → Main application script• dog_breed_model.tflite → TensorFlow Lite model for dog breed classification• cat_breed_model.tflite → TensorFlow Lite model for cat breed classification• dog_class_indices.pkl → Dog breed index mapping• cat_class_indices.pkl → Cat breed index mapping• Image/ → Reference images for breeds (optional for display)• assets/ → Icons and UI images (paw_logo.png, Dog_logo.png, Cat_logo.png, etc.)• db_faces.npy (expected) → Stored embeddings of authorized faces• shape_predictor_68_face_landmarks.dat → face detection• Cat_Breed_tune.ipynb → training notebook for cat• Dog_Breed_final.ipynb → training notebook for dogHow to Run?Ensure all required files (.tflite, .pkl, db_faces.npy, .dat and UI images) are available in the project folder.Run the program with -> `python test.py`The app will open with a face authentication page:After press start authentication button, the camera begin to capture your face.Embeddings are compared against stored reference embeddings.Need to pass the threshold value (>0.90) for five consecutives time to consider successAfter successful authentication, user is navigate to pet classification pageSelect Dog or Cat classification.Upload an image or capture using the camera.View the top 3 predicted breeds with confidence scores.NotesImage size for inference: 224×224 (RGB)Authentication:• Uses dlib for face detection.• Uses imgbeddings (EfficientNet-based CNN) for embedding extraction.• Matches embeddings against db_faces.npy.• Threshold (MATCH_THRESHOLD) controls strictness.Classification:• Based on pretrained TensorFlow Lite models for dogs and cats.• GUI resizes to 224×224 and normalizes to match training preprocessing.Quick check to verify TFLite input size:```pythondog_interpreter.get_input_details()[0]['shape']  # expect [1, 224, 224, 3]cat_interpreter.get_input_details()[0]['shape']  # expect [1, 224, 224, 3]```Training NotebooksCat_Breed.ipynbFramework: TensorFlow/KerasTransfer learning: Yes (EfficientNetB4, InceptionV3, Xception)Data augmentation: YesImage size: 224x224 (RGB)Batch size:
